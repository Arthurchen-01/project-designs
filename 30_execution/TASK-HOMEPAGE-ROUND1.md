## 具体改进

### 1. Hero 区域 — 从"广告感"到"压迫感"
- **之前:** 简单的标题+副标题，radial-gradient背景
- **之后:** 
  - 全屏Hero区域（min-height: 70vh），居中展示
  - 顶部badge标签：`EST. 2024 · 搏击战绩权威平台`
  - 标题改为更有力量感的文案：`用拳头书写每一页历史`
  - 副标题直接展示核心数字：`218 场比赛 · 178 场胜利 · 89 次 KO/TKO · 14 枚奖牌`
  - 双按钮CTA：实心金色「浏览战绩」+ 描边「核心选手」
  - 背景脉冲动画（heroPulse）

### 2. 统计数字区域 — 更大更有力
- 字号从 2.2rem → 2.8rem（桌面端），移动端 1.5-1.7rem
- 字重从 800 → 900
- 加入 `text-shadow` 金色微光
- 网格从 auto-fit → 固定6列（桌面）/ 3列（平板）/ 2列（手机）
- 加入 `box-shadow` 和 hover 效果

### 3. 重点比赛卡片 — 金色视觉突出
- `isKey=true` 的卡片自动获得 `match-key` class
- 效果：
  - 金色边框（rgba(212,168,67,.3)）
  - 渐变背景（左上角金色微光）
  - 顶部金线装饰（::before伪元素）
  - hover 时金色发光阴影
- 标签文字从"重点"改为"★ 重点"

### 4. 核心选手卡片 — "王牌"样式
- 首页4位选手获得 `fighter-ace` class
- 头像从 80px → 110px（ace）/ 96px（普通）
- 头像边框从 2px → 3px，加入金色box-shadow
- hover时头像放大 + 金光增强
- 背景加入微弱金色渐变
- 统计区域加了上分割线

### 5. 整体视觉优化
- Section 分区加入英文小标签（LATEST / ACES）
- Section 标题字号加大（2rem）
- 卡片 hover 从 translateY(-2px) → translateY(-3px) + 深阴影
- 查看全部链接改为带边框的按钮样式
- 导航栏加入 active 下划线指示器
- 关闭按钮加入 hover 效果
- Modal 背景加入 backdrop-filter: blur

### 6. Bug 修复
- 修复 fighter modal 中 `f.highlights`（数组）→ `f.highlight`（字符串）的引用错误

### 7. 移动端适配
- **1024px:** stats 改3列
- **768px:** 卡片单列，hero缩小，nav全屏下拉
- **480px:** stats 改2列，hero字号1.8rem

保持暗色系 `#0a0a0a` + 金色 `#d4a843` 主题。整体风格从"普通展示页"升级为"权威战绩平台"——厚重、有力、克制。没有花哨动效，靠排版、对比、间距营造压迫感。
