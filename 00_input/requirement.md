# 需求：苏格拉底家教系统的自动化与复习系统

- 提出者：GuanYu (ou_bd493a2c75313d6215d132564bf9f99a)
- 时间：2026-03-30
- 来源：飞书群聊

## 背景

用户已经有一个成熟的苏格拉底家教系统（Sakura 版），位于 `C:\Users\25472\Sakura - gemini版\`。
包含四位虚拟老师（三笠、明日香、小樱、剑心），系统设定完整，已有 5 节课的学习记录。

## 痛点

1. **每次新会话都需要手动喂设定**：打开 Gemini/GPT 浏览器侧边栏后，需要手动打开文件夹、粘贴角色设定等
2. **没有持久记忆**：对话结束后，下次就忘了上下文
3. **学习后无法复习**：没有办法自动生成 Anki 卡片或复习列表
4. **没有角色驱动的复习体验**：希望复习时能见到那个角色，用苏格拉底方式把学过的问题再问一遍

## 需求

### 核心需求
1. 自动化启动流程：不需要每次都手动喂设定
2. 持久化记忆：跨会话的记忆保持
3. 学习后自动生成复习材料（Anki 格式或复习列表）
4. 角色驱动的复习体验：复习时虚拟老师用苏格拉底方式重新提问

### 复习系统需求
- 方案 A：导出 Anki 卡片（.apkg 或 .csv 格式）
- 方案 B：自建复习系统（网站/工具），记录对话，复习时让角色重新提问

## 用户提到的参考

1. Q&A 学习法（知乎文章）：看书时记 Q，看完后找 AI 解答，复习时看 Q 回忆 A
2. "苏格拉底·七"系统（吴乐旻的知乎文章）：用 Claude Code 搭建的苏格拉底家教系统
