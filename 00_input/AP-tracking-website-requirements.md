# AP备考追踪网站项目需求文档

## 项目概述
一个AI驱动的AP考试备考追踪平台，帮助班级学生可视化学习进度、预测5分概率，并提供资源分享功能。

## 核心功能模块

### 1. 首页 - 班级选择器
- 设计为可扩展架构，支持多个班级
- 初期仅开放当前班级
- 点击班级进入对应仪表盘

### 2. 班级仪表盘（第一层）
#### 数据概览模块（可点击深入）：
1. **全班报名AP科数统计**
2. **班级总体5分概率**
3. **历年模考平均MCQ得分率**
4. **历年模考平均FRQ得分率**

#### 5月考试日历：
- 显示整个5月的考试安排
- 每个日期方块包含：
  - 距离今天的天数
  - 考试科目人数
  - 班级平均5分概率
- 设计要求：方块较大，颜色清晰，便于查看

### 3. 数据深入层（第二层）
点击班级仪表盘的任一模块后：
- **报名科数模块**：显示所有人姓名+报名科目的表格
- **5分率模块**：显示每个人的5分概率
- **点击某个人**：进入个人中心

### 4. 个人中心（第三层）
#### 个人总览：
1. FRQ测试情况
2. MCQ测试情况  
3. 计时vs不计时对比
4. 整体5分概率+趋势图
5. 报名科目数+各科5分概率（可点击）
6. 知识点掌握度（总体）

#### 点击某科后：
- **上方**：该科5分率趋势、FRQ/MCQ历次成绩、计时统计
- **下方**：按单元分的知识点掌握度（如AP Macro的Unit 1-10）

## AI驱动功能

### 5分率计算模型
#### 数据输入：
- 测试成绩（MCQ/FRQ，计时/不计时）
- 复习描述（文字描述复习内容）
- 历史表现记录

#### AI评估逻辑：
1. **有测试成绩**：历史表现占60% + 近期趋势占40%
2. **有复习描述**：根据描述深度给分
   - 越具体增幅越大
   - 仅说"复习了"三个字几乎不加分甚至微降
3. **什么都不填**：直接衰减（记忆曲线效应）
4. **稳定高分者**：边际收益递减，模糊输入基本不影响

#### 每日更新表单：
学生需要填写：
- 活动类型（FRQ测试/MCQ测试/复习/其他）
- 分数（如有测试）
- 具体描述（AI根据描述深度评估）

### 知识点掌握度评估
#### V1版本（先实现）：
AI根据分数整体推测每个单元的掌握度

#### V2版本（未来实现）：
AI逐题分析原卷，定位到具体知识点

## 待解决问题

### 1. 5分率计算公式
- 需要基于历年AP考试数据建立基准
- 确定AI评估的具体权重分配

### 2. 数据维度分类
- MCQ/FRQ分类标准
- 计时/不计时场景
- 复习/刷题区分

### 3. 效果量化
- 知识点复习对5分率的提升程度
- 需要历史数据支持

### 4. AI Prompt设计
- 如何让AI稳定评估学生水平
- 如何处理模糊描述（如仅说"复习了"）
