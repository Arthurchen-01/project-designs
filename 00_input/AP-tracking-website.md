## 一、项目背景

班里同学都在备考AP（5月份考试），需要一个**可视化进度追踪平台**，让同学和老师能实时看到：
- 距离目标还有多远
- 5分（满分）的成功概率有多高
- 每天的复习是否有效果

这个项目**可以推广给其他班级使用**，设计时要考虑通用性。

---

## 二、网站整体架构

```
Landing Page（首页 - 班级选择）
  └── [当前仅开放一个班级]
       └── 班级仪表盘 (Class Dashboard)
            ├── 📊 Dashboard（班级概览）← 默认标签页
            │   ├── 模块1：全班AP报名总科数
            │   │   └── [点击] → 全员报名清单（姓名 + 科目）
            │   ├── 模块2：总体5分概率
            │   │   └── [点击] → 每人5分率排名表
            │   │        └── [点击某人] → 该生个人仪表盘
            │   ├── 模块3：历年MCQ平均得分率
            │   │   └── [点击] → 每人MCQ得分率详情
            │   └── 模块4：历年FRQ平均得分率
            │        └── [点击] → 每人FRQ得分率详情
            │
            ├── 📅 考试日历（Calendar）← 五月
            │   ├── 每天方块较大，颜色清晰（方便老师查看）
            │   ├── 每日显示：
            │   │   ├── 距今天数
            │   │   ├── 当天考试科目 + 报名人数
            │   │   └── 该科目班级平均5分概率
            │   └── 点击具体科目 → 报名该科的学生列表
            │
            └── 👤 个人中心（Personal）
                ├── 个人仪表盘
                ├── 个人5分概率（多维度计算）
                ├── MCQ / FRQ 得分趋势
                ├── 知识点掌握进度
                └── 每日复习效果追踪
```

## 三、页面详细设计

### 3.1 首页（Landing Page）

- 功能：选择/进入班级
- 当前状态：先只开放「我们班」
- 设计：简洁，一个班级入口按钮即可

### 3.2 班级仪表盘（Dashboard 标签页）

| 模块 | 展示内容 | 点击后 |
|------|----------|--------|
| 报名总科数 | 全班AP报名科数总和 | 全员报名清单（姓名+科目，多行两列） |
| 总体5分概率 | 班级平均5分概率 | 每人5分率排名表 |
| MCQ平均得分率 | 历年模考MCQ平均得分 | 每人MCQ详情 |
| FRQ平均得分率 | 历年模考FRQ平均得分 | 每人FRQ详情 |

**模块设计**：仪表盘上是一个个**可点击的方块/卡片**，呈现汇总数据。点击后进入对应的明细表格。

**嵌套导航逻辑（关键）**：
```
班级仪表盘 → 点击「5分率」模块
  → 每人5分率排名表
    → 点击某人
      → 该生个人仪表盘（独立页面）
```

### 3.3 考试日历（Calendar 标签页）

- 时间范围：仅5月
- 每天方块比常规日历大
- 颜色编码要清晰（区分不同科目/状态）
- 每个方块包含：
  - 考试日期
  - 距离今天的天数
  - 当天考试科目
  - 班级报名该科的人数
  - 该科班级平均5分概率

### 3.4 个人中心（Personal 标签页）

- 每个学生登录/选择自己后进入
- 包含个人独立仪表盘（从班级5分率点进来也能到此）
- 个人维度的数据追踪

## 四、待解决问题（核心难点）

### 4.1 5分概率计算模型 ❌ 未确定

- 需要确定具体的计算逻辑和公式
- 初步想法：模考75分以上 → 高5分置信度（但具体等级划分待定）
- 可能需要多维度综合计算（不只是单次模考分数）

### 4.2 数据维度定义 ❌ 待明确

需要区分以下维度来记录每次复习/练习：

| 维度 | 问题 |
|------|------|
| 测试类型 | MCQ 还是 FRQ？ |
| 计时模式 | 计时的还是不计时的？ |
| 任务类型 | 知识点复习 vs 刷题 |
| 知识点粒度 | 具体到哪个知识点？ |

### 4.3 效果评估 ❌ 难以量化

- 知识点复习后，对5分概率的**具体提升程度**难以估计
- 这也是老师最关心的：今天的复习是否对提升5分率有显著效果
- 需要设计一个合理的评估模型

### 4.4 数据收集方式

- 每人每天填写/更新初始信息
- 系统自动记录每日进展变化
- 需要设计简洁的每日录入表单
